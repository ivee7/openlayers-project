## Openlayers Project

### Используемый стек:

- Vue 3 Composition API;
- TypeScript;
- Openlayers;
- Pinia;
- SCSS;
- Primevue.

### Установка:

```js
npm install
```

### Команда для запуска в dev-режиме:

```js
npm run dev
```

### Команда для build-сборки:

```js
npm run build
```
